# 🏆 IBU Software – Dart Turnier Verwaltungs Tool

Desktop-Anwendung (PyQt6) zur Verwaltung von Dart-Turnieren und Meisterschaften.  
Läuft lokal auf Windows, speichert Daten in SQLite.

---

## 🚀 Schnellstart

```bash
# Im Projektordner
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install PyQt6

# Start
python main.py
